About RED's Mighty Mouse Kit

This kit includes many menu, including a large collection of pie menus, designed to speed up workflows in Modo. I used to find myself digging through different tabs and menus to find the tools or settings I wanted. I switched to a 12 button mouse years ago and started slowing mapping it for increased productivity. I took many of the best tools in modo and made them easily assessible with a keystroke and a swipe of the mouse. Then I started adding cutsom hot keys for specific tools. I high recommend you map you left and right mouse with tilts to down and up respectively to take full advantage of these specific keybindings. I've also added an "Open with Default App" command to the right click menu, in the shader tree to make these easier to access.

    Keyboard Shortcuts

Ctrl-m = @RED_dupmat.py A script that takes the shader tree layers you have selected and duplicates them to a new material that it requests a name for. (This is really useful for splitting off materials into separate groups to re-color parts.)

Ctrl-shift-m = Edit Selection Set Dialog Box

Shift+P = Fill with Quads (Grid)
